DCRE: Distribution-Conditional Retrieval Embedding for Aligning LLM Agents with Statistical Ecosystems

Abstract

Large Language Model (LLM)-based agents have shown strong capabilities in automating data science workflows, yet many rigorous and domain-specific statistical methods remain underutilized due to limitations in model knowledge and tooling. Existing Retrieval-Augmented Generation (RAG) approaches rely primarily on function-level semantic similarity and fail to account for data distribution characteristics that critically determine the applicability of statistical methods, leading to inaccurate retrieval and inefficient use of statistical resources. To address this limitation, we propose DCRE (Distribution-Conditional Retrieval Embedding), a distribution-aware retrieval model that conditions function representations on data distribution profiles. Our contributions are threefold: (i) we construct a curated knowledge base of 8,191 high-quality R package with functions spanning diverse statistical domains; (ii) we introduce DCRE, an embedding model that explicitly incorporates data distribution information for accurate R package retrieval; and (iii) we design URTABench, the first benchmark for evaluating LLM agents on R-based data analysis tasks. Experimental results show that DCRE achieves an NDCG@10 of 0.9347, outperforming open-source state-of-the-art embedding models by up to 17% while using only 23M parameters. Furthermore, integrating DCRE into LLM agents significantly improves downstream statistical analysis performance up to 62.5% on URTABench.
