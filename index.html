DCRE: Distribution-Conditional Retrieval Embedding for Aligning LLM Agents with Statistical Ecosystems

Anonymous submission
[Paper PDF] [Code & Data]

Abstract

Large Language Model (LLM)-based agents have shown strong capabilities in automating data science workflows, yet many rigorous and domain-specific statistical methods remain underutilized due to limitations in model knowledge and tooling. Existing Retrieval-Augmented Generation (RAG) approaches rely primarily on function-level semantic similarity and fail to account for data distribution characteristics that critically determine the applicability of statistical methods, leading to inaccurate retrieval and inefficient use of statistical resources. To address this limitation, we propose \textbf{DCRE} (\textbf{D}istribution-\textbf{C}onditional \textbf{R}etrieval \textbf{E}mbedding), a distribution-aware retrieval model that conditions function representations on data distribution profiles. Our contributions are threefold: (i) we construct a curated knowledge base of \textbf{8,191 high-quality R functions} spanning diverse statistical domains; (ii) we introduce DCRE, an embedding model that explicitly incorporates data distribution information for accurate R package retrieval; and (iii) we design \textbf{URTABench}, a benchmark for evaluating LLM agents on R-based data analysis tasks. Experimental results show that DCRE achieves an \textbf{NDCG@10 of 0.9347}, outperforming open-source state-of-the-art embedding models by \textbf{up to 17\%} while using only \textbf{23M parameters}. Furthermore, integrating DCRE into LLM agents significantly improves downstream data science performance on URTABench.
